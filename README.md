##  Configuração inicial do projeto
As instruções deste guia devem ser executadas em um terminal do Git Bash:

Preparatório:
Com o projeto aberto no VSCode, abra um terminal do Git Bash e digite os comandos abaixo:
(para baixar as instruções da prova)
1. git clone https://github.com/VicCAlq/2ano.git
(para baixar o material de apoio)
2. git clone https://github.com/VicCAlq/apoio.git

Para executar o projeto:
1. Abra outro terminal do Git Bash para rodar o cliente.
2. Instale as dependências do projeto utilizando o Node com o comando `npm install`
    - A lista de dependências se encontra no arquivo `package.json`
3. Execute o cliente com o comando `npm run dev`
4. Acesse o endereço `http://localhost:3000`

##  Para rodar novamente após encerrado

Dentro da pasta do projeto, execute os comandos abaixo em um terminal do Git Bash:

1. `npm install`
2. `npm run dev`

Tudo dando certo, o site estará acessível em `http://localhost:3000`
